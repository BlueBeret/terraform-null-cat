# terraform-null-cat

Infrastructure as Code without any paid resources.

There are several branches you can explore:

- [`main`](https://github.com/BlueBeret/terraform-null-cat/tree/main): Basic Terraform setup with input and output variables.
- [`many-resources`](https://github.com/BlueBeret/terraform-null-cat/tree/many-resources): Multiple Terraform resources (e.g., local, null) — all free.
- [`many-outputs`](https://github.com/BlueBeret/terraform-null-cat/tree/many-outputs): A variety of Terraform output examples.
